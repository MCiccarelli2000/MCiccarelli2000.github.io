# Web Mauro Ciccarelli · Chef privado a domicilio

Versión estática gratuita lista para publicar en Netlify, GitHub Pages o cualquier hosting de HTML.

## Cambios incluidos

- Menú de 65 € renombrado como **Menú Selección**.
- Foto de pasta fresca enlazada por el usuario incorporada.
- Se mantiene **una sola foto de brasa/parrilla** para evitar duplicidad.
- Se añade plato de **pescado dorado por la piel**.
- Se añade plato de carne/cordero.
- Se eliminan fotos de postre de la galería.
- Galería final con 7 fotos seleccionadas.

## Antes de publicar

Sustituir en `index.html`:

- `TUEMAIL@DOMINIO.COM`

Completar aviso legal, privacidad y condiciones de contratación antes de usarla comercialmente.

## Publicación gratis en Netlify

1. Entra en Netlify.
2. Crea un sitio nuevo.
3. Arrastra la carpeta `mauro-chef-privado-final-web` o el ZIP descomprimido.
4. Revisa que el formulario de contacto quede activo.
